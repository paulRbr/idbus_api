## Description

This gem wraps the new iDBUS API v1 available at https://api.idbus.com

## Installation

You can use this gem in your project by specifying it in your `Gemfile`:

```
gem "idbus_api"
```

or simply install it via the CLI:

```
gem install idbus_api
```
